## 🚀 Basic Kali Linux Commands
Here are some fundamental commands every Kali Linux user should know:

### 📂 File Management
- `ls` – List files and directories
- `cd <directory>` – Change directory
- `pwd` – Show current directory path
- `cp <source> <destination>` – Copy files
- `mv <source> <destination>` – Move/Rename files
- `rm <file>` – Remove files

### 👤 User Management
- `whoami` – Show the current user
- `sudo su` – Switch to root user
- `adduser <username>` – Add a new user

### 🌐 Networking
- `ifconfig` or `ip a` – Show network interface details
- `ping <website>` – Check network connectivity
- `netstat -tulnp` – Display active connections

### 🔍 System Monitoring
- `ps aux` – Show running processes
- `top` or `htop` – Display system resource usage
- `kill <PID>` – Terminate a process

### 🔐 Permissions & Ownership
- `chmod <permissions> <file>` – Change file permissions
- `chown <user>:<group> <file>` – Change file ownership
